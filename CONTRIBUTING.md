# Contributing to Awesome Blockchain & Greentech

First off, thanks for taking the time to contribute!

The following is a set of guidelines for contributing to this awesome list.

## How to contribute

1.  **Fork the repository** on GitHub.
2.  **Create a new branch** for your feature or addition (`git checkout -b add-amazing-project`).
3.  **Add your link** to the appropriate section in `README.md`.
    *   Please follow the format: `- [Project Name](Link) - Short, objective description.`
    *   Keep descriptions concise.
    *   Ensure the link is valid and points to the official repository or documentation.
4.  **Commit your changes** (`git commit -am 'Add Amazing Project'`).
5.  **Push to the branch** (`git push origin add-amazing-project`).
6.  **Create a new Pull Request**.

## Guidelines

*   **Quality**: Ensure the project is active, well-documented, and useful.
*   **Order**: Alphabetical order within categories is preferred unless ranking by popularity/relevance makes more sense contextually (currently generic ordering).
*   **No duplicates**: Check if the link is already in the list.
